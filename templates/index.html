Romanized Telugu to English Translator

This project focuses on enabling seamless translation of Romanized Telugu (Telugu written in Latin script) into English. It first uses AI4Bharat's XlitEngine for accurate transliteration to native Telugu script, followed by translation using a Fine-Tuned Version of "Helsinki-NLP/opus-mt-mul-en" model which is trained to take inputs from any kind of language and give output in English. This tool aims to bridge the gap for Telugu speakers who frequently type in Roman script but seek high-quality English translations for professional, educational, and everyday use cases.

We trained and evaluated the system on benchmark datasets and used robust metrics like BLEU, METEOR, ROUGE, chrF, BERTScore, COMET, and TER to ensure high-quality outputs in both direct Telugu and Romanized Telugu inputs.
